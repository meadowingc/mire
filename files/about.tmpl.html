about

This is a fork of the excelent vore.website (source) RSS feed reader. You can find the source code for mire on Codeberg. Any contributions are welcome!

mire aims to be a minimalistic RSS reader in the same vein as vore, but with slightly more features. I originally created the fork because I wanted something exactly like vore but with the ability to know if I had already read an article, as well as maybe also having the ability to get notifications when new articles are posted. I opted for forking the project instead of contributing the changes back to the main project because I saw it was part of vore's design not to have these features in the first place.
